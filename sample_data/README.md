# GeoRoute-Net Mock Satellite Sandbox

This directory contains pre-processed 512x512 mock data patches to evaluate the Multiplicative Graph Healing framework without massive geospatial download dependencies.

- `input/`: Contains pre-tiled binary road masks mimicking canopy occlusion breaks.
- `output/`: Holds the generated routable topological graphs, `.geojson` vector overlays, and JSON feedback logs.
